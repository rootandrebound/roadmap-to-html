Which one is right for me ‚Äî a State ID Card or a Driver License? What‚Äôs the difference?

Both a California state ID and a California driver license serve as an official government-issued, photo identification that can be used to prove your identity, age, and legal presence in the United States (unless you have an ‚Äúundocumented person‚Äù California driver license). Either one will allow you to prove your identity, for example, when you open a bank account, register to vote, or apply for jobs, housing, or public benefits.

If you eventually want to drive, you will need to get a driver license, but because that process requires testing, we recommend that you get a California state ID first, since ID is needed right away after release. Once you get a state ID, you can go back later for a driver license.
